# 🎨 NexCart – UI/UX Design Document
# Inspired by: Piccollo Backpack Landing Page (Dribbble)

---

## 📌 1. Design Philosophy

The reference design (Piccollo landing page) follows:

- Minimalist layout
- Strong product focus
- Smooth motion/animation
- High whitespace usage
- Premium feel

👉 NexCart will adopt this style to:
- Enhance user engagement
- Make AI feel “premium + intelligent”
- Simulate modern e-commerce UX

---

## 🎯 2. Design Goals

- Clean and modern UI
- Product-first storytelling
- Smooth micro-interactions
- Scroll-based engagement
- Visual hierarchy clarity
- AI recommendation feels “smart, not forced”

---

## 🎨 3. Visual Style Guide

---

### 🎨 Color Palette

Primary:
- Soft neutral background (Off-white / Light beige)

Accent:
- Deep black / charcoal (text)
- Subtle pastel tones (cards, highlights)

CTA Color:
- Dark solid (Black / Deep Green)

---

### 🔤 Typography

- Headings: Bold Sans-serif (e.g., Inter / Poppins)
- Body: Clean readable font

Hierarchy:
- H1: Large, bold (Hero section)
- H2: Section titles
- Body: Minimal text blocks

---

### 🧱 Layout Principles

- Grid-based structure
- High spacing (padding-heavy)
- Center-aligned hero content
- Horizontal product scroll sections

---
